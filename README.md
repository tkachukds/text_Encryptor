Это шифровальшик текста. Сделан на основе Цезарь сдвига. 

Почему это хороший шифровальщик? 
Допустим есть текст для шифровки - 
Введите текст для шифрования: Привет, это супер важный текст, который не должен быть расшифрован
Введите коды сдвига через запятую: 19,16,4,5,3,6,13,16,17,19,3,4,25,19,12,9,3,4,5,6,7,8

Через запятую, вы вносите сдвиг предыдущего результата. 

Сначала текст сдвигается на 19
Потом полученный результат сдвигается на 16
Потом полученный текст сдвигается на 4
и т д.

Итог:
Зашифрованный текст: ииаыюк, хкж йлзюи ыщяеуб кювйк, вжкжиуб ею эжгяюе ъукф ищйрамижыще
Выберите действие (1 - Зашифровать, 2 - Расшифровать, 0 - Выйти): 2
Введите текст для расшифровки: ииаыюк, хкж йлзюи ыщяеуб кювйк, вжкжиуб ею эжгяюе ъукф ищйрамижыще
Введите коды сдвига через запятую: 19,16,4,5,3,6,13,16,17,19,3,4,25,19,12,9,3,4,5,6,7,8
Расшифрованный текст: рривет, это супер важный текст, который не должен быть расшифрован
